# Amazon-Home-Page
A front-end Amazon Clone developed using HTML5 and CSS3. Features include a responsive navigation bar, search box, hero banner, product showcase cards, and Amazon-style footer. Created as a BCA web development project to practice modern UI design and layout techniques.
# Amazon Clone

This project is a clone of the Amazon homepage built using HTML and CSS.

## Features
- Amazon-style Navbar
- Search Bar
- Language Selector
- Hero Section
- Product Cards
- Responsive Layout
- Multi-section Footer

## Technologies Used
- HTML5
- CSS3
- Font Awesome

## Purpose
This project was created for learning front-end web development and improving HTML/CSS skills as part of a BCA college project.
